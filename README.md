# Task Manager
Manage People, Manage Tasks, Assign to people, and get notified

## Tech Stack
- Next.js
- Prisma
- MySQL

## Installation

### Next.js
`npx create-next-app@latest`

### Prisma
`npm install prisma @prisma/client`
`npx prisma init`
`npx prisma migrate dev --name initial_migration`
`npx prisma generate`

### Chart.js
`npm install chart.js react-chartjs-2`
